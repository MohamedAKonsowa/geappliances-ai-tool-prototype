Create a professional, clean, and functional employee evaluation app that accepts a CSV file containing time sheet data and returns a comprehensive evaluation report, adhering to GE Appliances branding guidelines. The app will feature a simple and intuitive interface, ensuring seamless user experience across various devices and screen sizes.

The app will include the following key features:
- A file upload component, utilizing a clear call-to-action button and a progress bar to display the upload status, allowing users to upload a CSV file.
- A data validation component that checks the uploaded CSV file for errors, such as missing or duplicate columns, and displays an error message if the file is invalid, ensuring data consistency and accuracy.
- A data processing component that reads the uploaded CSV file and calculates the total hours worked, average hours worked per day, and other relevant metrics for each employee, providing valuable insights into employee performance.
- A data visualization component that displays the calculated metrics in a clear and concise manner, using charts and graphs, such as bar charts or line charts, to help users quickly understand the data and make informed decisions.
- A report generation component that generates a comprehensive evaluation report for each employee, including their total hours worked, average hours worked per day, and other relevant metrics, enabling users to assess employee performance effectively.

- A report component that displays the generated evaluation report for each employee, including their total hours worked, average hours worked per day, and other relevant metrics, facilitating effective employee evaluation.
